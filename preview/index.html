Game Scenes

Main Menu

Western desert background with character selection (Entrepreneur/Adventurer) and game start options.

Key Scripts:
- MainMenuController.cs
- CharacterSelection.cs

Daytime Ranch (Cipher Gulch)

Manage resources: breed cattle (NFTs), upgrade barn capacity, and handle day-to-day operations.

Key Features:
- Resource Management (Hay, Water)
- Cattle Breeding (Speed, Milk traits)
- Barn Upgrades

Saloon

Place wagers in Cipher Poker with 50/50 chance, 10% burn, and 2x potential winnings.

Key Mechanics:
- Wager: 1-50 $CATTLE
- Win: 2x wager after burn
- Lose: Wager lost

Nighttime Ether Range

Craft Shadow Potions (NFTs) using $CATTLE and sell at fluctuating market prices.

Key Mechanics:
- Potion Cost: 20 $CATTLE (50% burn)
- Random Potency trait
- Selling Price: 25-35 $CATTLE

Educational Elements

  • Ranch: Budgeting and resource allocation
  • Saloon: Risk management and probability
  • Ether Range: Supply/demand market dynamics
  • Sticky Notes: Financial tips and reminders

Technical Implementation

The project is implemented with:

  • Unity 2D with pixel-perfect camera (16x16 pixel units)
  • C# scripting for game mechanics
  • Mock blockchain integration (ready for Solana in future)
  • Day-night transition system
  • Educational tooltip system
  • Sticky note generation system (for future printer integration)
