Game Mechanics with Blockchain Integration

Daytime: Cipher Gulch

  • Resource Management: Hay and Water resources
  • Cattle Breeding: Creates NFTs on Solana with unique traits
  • Barn Upgrades: Spend $CATTLE to increase capacity
  • Wallet Integration: Connect to Phantom, Solflare, or Backpack

Nighttime: Ether Range

  • Shadow Potions: Craft NFTs with $CATTLE (50% burn)
  • Market Dynamics: Fluctuating prices based on market conditions
  • NFT Trading: Sell potions for $CATTLE tokens
  • MPL-404 Conversion: Convert between NFTs and tokens

Saloon Gambling

The Saloon features a blockchain-integrated poker game with token burning mechanics:

  • Wager 1-50 $CATTLE tokens on a 50/50 chance
  • 10% of wager is permanently burned from token supply
  • Win = 2x remaining wager amount (after burn)
  • All transactions execute on-chain with Solana
  • Financial education about risk management and probability

Sticky Note System

Educational financial tool with blockchain integration:

  • Generate sticky notes with financial/blockchain tips
  • Option to mint notes as NFTs on Solana
  • Three templates: To-Do List, Financial Tip, Blockchain Tip
  • Integrates real player data from wallet and game state
  • Prepared for future physical printer integration

MPL-404 Token Standard Implementation

The game leverages Metaplex's MPL-404 "Hybrid" token standard which allows seamless conversion between:

  • NFT Mode: Tokens exist as unique NFTs with attributes (Cattle with Speed/Milk traits)
  • Fungible Mode: Tokens can be converted to $CATTLE based on attributes
  • Conversion Benefits: Players can liquidate NFTs when they need currency or create unique NFTs when they want to invest
  • Economic Education: Teaches concepts of liquidity, asset value, and market dynamics
